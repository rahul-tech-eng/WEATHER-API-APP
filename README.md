# 🌦️ Weather App

A simple and responsive Weather App built using **HTML, CSS, and JavaScript**. It fetches real-time weather data from a Weather API and displays the current weather conditions of any city entered by the user.

## 📌 Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature
- 💧 Shows humidity
- 💨 Displays wind speed
- 🌥️ Weather condition with icons
- ❌ Error message for invalid city names
- 📱 Fully responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API (OpenWeatherMap API)
